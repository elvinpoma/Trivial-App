# Trivial-App

Trivial-App es un juego de preguntas y respuestas desarrollado en Kotlin y Jetpack Compose. La aplicación ofrece una experiencia sencilla y atractiva para poner a prueba tus conocimientos en informática, con retroalimentación visual inmediata y resultados finales al terminar el juego.

## Descripción de la app

- Pantalla de inicio con el menú principal para comenzar el juego.
- Sistema de preguntas con respuesta correcta o incorrecta.
- Temporizador y barra de progreso que cambia de color cuando el tiempo se agota.
- Pantalla final con el resultado total.
- Opción para compartir los resultados al finalizar.
